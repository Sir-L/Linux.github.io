# Linux笔记

​				——Linux命令

## poweroff

#### 说明

​		用于关闭计算机操作系统并且切断系统电源

#### 语法

​		poweroff [Option]

| Option |                             说明                             |
| :----: | :----------------------------------------------------------: |
|   n    |                 关闭操作系统时不执行sync操作                 |
|   w    |      不真正关闭操作系统，仅在日志文件“/var/log/wtmp”中       |
|   d    | 关闭操作系统时，不将操作写入日志文件“/var/log/wtmp”中添加相应的记录 |
|   f    |                       强制关闭操作系统                       |
|   i    |              关闭操作系统之前关闭所有的网络接口              |
|   h    |       关闭操作系统之前将系统中所有的硬件设置为备用模式       |
