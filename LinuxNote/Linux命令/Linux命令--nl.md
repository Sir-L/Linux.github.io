## nl

#### 说明

​		用来计算文件中行号

#### 语法

​		nl [Option] [文件]

| Option |          说明           |
| :----: | :---------------------: |
|  -b a  |      空行显示行号       |
|  -b t  |     空行不显示行号      |
| -n ln  |     行号最左边显示      |
| -n rn  | 行号最右边显示，且不加0 |
| -n rz  |  行号最右边显示，且加0  |
