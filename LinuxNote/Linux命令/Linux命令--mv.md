# Linux笔记

​				——Linux命令

## mv

#### 说明

​		用来移动 (改名) 文件

#### 语法

​		mv [Option] 源文件 目标文件

| Option |              说明              |
| :----: | :----------------------------: |
|   -f   |         覆盖前永不提示         |
|   -i   |           覆盖前提示           |
|   -u   | 只移动更老的或者标记新的非目录 |
|   -v   |          说明完成过程          |
