# Linux笔记

​				——Linux命令

## passwd

#### 说明

​		用于更改使用者的密码

#### 语法

​		passwd [Option]

| Option |                 说明                 |
| :----: | :----------------------------------: |
|   -d   |               删除密码               |
|   -f   |               强制执行               |
|   -k   | 保留即将过期的用户在期满后能仍能使用 |
|   -l   |               禁用账户               |
|   -S   |             显示密码信息             |
|   -u   |           启用被禁用的账号           |
|   -x   |            设置密码有效期            |
|   -g   |             修改群组密码             |
|   -i   |     密码过期相应天数后，账号禁用     |
|   -x   |        两次密码修正的最大天数        |
|   -n   |        两次密码修改的最小天数        |
|   -w   |      在距多少天提醒用户修改密码      |
