# Linux笔记

​				——Linux命令

## cd

#### 说明

​		用于切换当前工作目录至目标目录

#### 语法

​		cd Option 目标目录(可为绝对路径或相对路径)

| 目标目录 |           说明           |
| :------: | :----------------------: |
|   省略   |    使用者的/home目录     |
|    ~     |    使用者的/home目录     |
|    .     |       目前所在目录       |
|    ..    | 目前所在目录的上一层目录 |
|    -     |   最近两次目录之间切换   |
