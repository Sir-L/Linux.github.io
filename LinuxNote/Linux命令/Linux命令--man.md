# Linux笔记

​				——Linux命令

## man

#### 说明

​		用于格式化并显示在线帮助手册页

#### 语法

​		man [Option] 关键字

| Option |                             说明                             |
| :----: | :----------------------------------------------------------: |
|   -a   |                        显示所有匹配项                        |
|   -w   | 不带搜索title 打印manpath变量 带title关键字 打印找到手册文件路径 |
