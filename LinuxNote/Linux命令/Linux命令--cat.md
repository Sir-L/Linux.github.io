# Linux笔记

​				——Linux命令

## cat

#### 说明

​		连接文件并在标准输出上打印

#### 语法

​		cp Option 文件位置

| Option |                 说明                  |
| :----: | :-----------------------------------: |
|   -A   |               等于-vET                |
|   -b   |           给非空输出行编号            |
|   -e   |  覆盖已经存在的目标文件而不给出提示   |
|   -E   |           在每行结束时显示$           |
|   -n   |           给所有输出行编号            |
|   -s   | 将所有的连续的多个空行替换为一个空行  |
|   -t   |                等于-vT                |
|   -T   |           把TAB字符显示为^I           |
|   -u   |          在复制前打印文件名           |
|   -v   | 除了LFD和TAB之外所有控制符用^和M-显示 |
