## mkfs

#### 说明

​		用于在特定的分区上建立 linux 文件系统

#### 语法

​		mkfs [-V] [-t 系统类型] [Option] 分区

| Option |                  说明                  |
| :----: | :------------------------------------: |
|   -V   |              显示处理过程              |
|   -t   |           给定档案系统的型式           |
|   -c   | 在制做档案系统前，检查该分区是否有坏轨 |
