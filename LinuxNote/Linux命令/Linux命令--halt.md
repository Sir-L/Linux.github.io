# Linux笔记

​				——Linux命令

## halt

#### 说明

​		用于关闭系统，但只是关闭系统，不掉电

#### 语法

​		halt [Option]

| Option |                        说明                         |
| :----: | :-------------------------------------------------: |
|   -n   |        在关机或重启之前不对系统缓存进行同步         |
|   -w   | 并不会真的关机，只是把记录写到 /var/log/wtmp 档案里 |
|   -d   |     不记录此次关机情况。当使用 -n 参数时隐含 -d     |
|   -i   |   强制执行 halt 或 reboot 而不去调用 shutdown(8)    |
|   -p   |   当关机的时候，顺便做关闭电源（poweroff）的动作    |
|   -f   |        在关闭或重启系统之前关闭所有网络界面         |
